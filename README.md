# O2 Agency - Landing Page Build PC & Marketing

Dự án Landing Page được thực hiện tại **O2 Agency** nhằm mục đích giới thiệu dịch vụ Build PC High-end và các giải pháp Digital Marketing.

### 🛠 Công nghệ sử dụng:
- **Frontend:** HTML5, CSS3 (Tailwind CSS CDN).
- **Animation:** AOS Library (Animate On Scroll).
- **Fonts:** Plus Jakarta Sans (Google Fonts).
- **Deployment:** Vercel / GitHub Pages.

### 🚀 Tính năng nổi bật:
- [x] Giao diện Responsive (Mobile-first).
- [x] Hiệu ứng Glassmorphism & Neon Glow hiện đại.
- [x] Tối ưu hóa Technical SEO & Performance.
- [x] Cấu hình tách biệt CSS/JS sạch sẽ.
